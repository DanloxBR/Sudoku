## 📘 Sobre o Projeto

Este é um jogo **Sudoku completo**, implementado em **Java**, utilizando conceitos essenciais como:

- Estruturação de classes  
- Organização por modelos (model), utilitários (util) e lógica principal  
- Enums para status do jogo  
- Validação de linhas, colunas e blocos  
- Impressão amigável do tabuleiro  
- Sistema de interação via terminal  

O objetivo foi **estudar, praticar e evoluir** como desenvolvedor, criando um projeto funcional e bem estruturado.

---

## 🛠️ Tecnologias e Conceitos Utilizados

- ☕ **Java 8+**
- 🧠 **Programação Orientada a Objetos**
- 📦 **Modelos e Enums**
- 🧪 **Validação lógica do Sudoku**
- 🎛️ **Scanner para leitura de comandos**
- 🧹 **Boas práticas e organização por pacotes**
